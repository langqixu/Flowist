# 脚本模板：睡前放松引导

## 使用场景
用户表达失眠、难以入睡、躺在床上思绪纷飞。

## 核心结构
1. **连接 (Connect)**：确认用户的困扰，建立共情
2. **过渡 (Transition)**：从日间模式切换到夜间模式
3. **身体放松 (Body Relaxation)**：渐进式肌肉放松
4. **思维平静 (Mind Calming)**：使用隐喻或呼吸
5. **结束 (Closure)**：温柔地引导进入睡眠

## 模板示例

### 开场（1-2 分钟）
```
{user_name}，晚上好。[2s]
我知道躺在床上，脑子停不下来，是一件很辛苦的事。[3s]
但现在，这段时间属于你。[3s]
什么都不需要做，什么都不需要解决。[5s]
只需要...在这里...和我一起...慢慢放松。[5s]
```

### 身体放松（3-5 分钟）
```
首先，让我们关注你的呼吸。[3s]
不需要改变它，只是注意它。[5s]
吸气...[4s]...呼气...[5s]
很好。[3s]

现在，把注意力带到你的双脚。[5s]
如果那里有紧张，想象呼吸流经它们。[5s]
吸气时，温暖流入。[5s]
呼气时，紧张释放。[10s]
```

### 思维平静（使用隐喻）
```
你的思维现在可能还很活跃。[3s]
这很正常。[3s]
想象一条夜晚的溪流。[5s]
那些待办事项、担忧...都是溪流上的树叶。[5s]
看着它们...一片一片...漂走。[10s]
你不需要解决它们，不需要记住它们。[5s]
明天的你会处理。[3s]
现在，只是...放下。[10s]
```

### 结束（1-2 分钟）
```
做得很好。[3s]
你的身体正在变重，思维正在变慢。[5s]
像沉入一张柔软的云朵。[5s]
允许自己...慢慢...滑入睡眠。[10s]
晚安，{user_name}。[5s]
```

## 注意事项
- 语速要**极慢**，停顿要**足够长**
- 避免使用"睡觉"、"入睡"等直接指令（会产生压力）
- 使用"允许"、"让身体決定"等被动语言
- 总时长控制在 10-15 分钟
