# 脚本模板：工作压力释放

## 使用场景
用户表达工作压力大、焦虑、肩颈紧张、脑子停不下来。

## 核心结构
1. **验证情绪**：确认压力是真实且合理的
2. **建立边界**：这段时间是"暂停键"
3. **身体释放**：针对肩颈的放松
4. **认知重构**：用隐喻帮助放下控制
5. **重新连接**：找回内在的平静

## 模板示例

### 开场
```
{user_name}，[2s]
我听到了你的压力。[3s]
{根据 weather/time 建立连接，如："外面正在下雨，而你承受着内心的风暴。"}[5s]
在接下来的这段时间，允许自己暂时放下所有待办事项。[3s]
它们会等你的。[3s]
但现在，你需要先照顾自己。[5s]
```

### 身体释放（重点：肩颈）
```
工作的压力常常累积在肩膀上。[3s]
把注意力带到你的肩膀。[5s]
吸气时，耸起肩膀，靠近耳朵。[4s]
用力...更用力...[3s]
然后，突然放下。[2s]
呼气，让肩膀完全塌下来。[5s]
感受那种释放。[5s]

再来一次。[3s]
吸气，抬起...[4s]...呼气，放下。[5s]
很好。[5s]
```

### 认知重构（使用盒式呼吸 + 隐喻）
```
现在，让我们用呼吸重新找回控制感。[3s]
盒式呼吸，四个阶段，每个阶段四秒。[5s]

吸气...[4s]...屏息...[4s]...呼气...[4s]...屏息...[4s]
再来一次。[2s]
吸气...[4s]...屏息...[4s]...呼气...[4s]...屏息...[4s]

很好。[5s]
你的那些担忧、待办事项，就像天空中的云。[3s]
它们存在，但它们不是你。[5s]
你是观察云的天空本身。[5s]
广阔、平静、不动摇。[10s]
```

### 结束
```
当你结束这次练习，重新回到工作中时，[3s]
你会发现，你的肩膀更轻松了。[3s]
你的思维更清晰了。[3s]
该做的事情还在，但你准备好了。[5s]
深吸一口气。[5s]
慢慢睁开眼睛。[3s]
很好，{user_name}。[3s]
```

## 关键话术
- "压力是真实的，但这不意味着你需要一直携带它。"
- "你不需要解决所有问题，才能允许自己休息。"
- "这段时间是你的避风港。"
